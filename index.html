What is Front-end Web Development ?

Front-end web development is responsible for the look and feel of a website. This means how colors, type, icons, and images appear. Increasingly, front-end development has to account for how a website looks on all devices, from desktop to tablet to phone. Typical programming languages include HTML, JavaScript, and CSS. Front-end developers keep up with cutting-edge trends in web design and development to make sure websites are optimized for users and search engines —
 and with security best practices in mind.
